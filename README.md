# Address Book Application

A simple desktop address book application built with Python and Tkinter that allows users to store, view, edit, and delete contact information with the option to interact through a graphical interface.

## Features

- Add new contacts (name, phone number, email, address, etc.)
- View saved contacts in a list
- Edit existing contact details
- Delete contacts
- Search for existing contacts
- User-friendly GUI built with Tkinter
- CSV file storage for contacts backup in the case of accidentally closing the application prematurely

## Technologies used

- Tkinter (GUI)
- CSV file storage  
- Python3 

## How to Run

1. Make sure Python3 is installed on your system (for MAC users).
2. Clone or download this repository.
3. Navigate to the project folder.
4. For GUI, run the application using:

```bash
python3 gui.py
```
Otherwise, for terminal output:
```bash
python3 mainProject.py
```
